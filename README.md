# modern-java

## setup

- install SDKMAN!
- install JDK 26
- install Maven
- generate new project with Maven archetype: 
  - `mvn archetype:generate -DgroupId=dev.shawngarner -DartifactId=modern-java -DarchetypeArtifactId=maven-archetype-quickstart -DarchetypeVersion=1.5 -DinteractiveMode=false`

## build

- mvn clean compile
- mvn clean test
